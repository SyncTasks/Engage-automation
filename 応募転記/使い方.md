# エンゲージ応募転記 自動実行システム

エンゲージ管理画面にログインし、新規応募者を自動でスプレッドシートに転記・通知送信します。

## 機能

- **自動ログイン**: Playwright + Chrome でエンゲージ管理画面にログイン
- **reCAPTCHA対応**: チェックボックスクリック → 2Captcha API で画像チャレンジ解決
- **応募者転記**: 応募者情報をスプレッドシートに自動転記（重複チェック付き）
- **通知送信**: Chatwork / Slack / LINE Notify に新規応募を通知
- **実行履歴**: 実行履歴スプレッドシートにクライアントごとの処理結果を記録

## セットアップ

### 1. 初期セットアップ

`初期セットアップ.bat` をダブルクリックして実行します。

### 2. 必要ファイルの配置

| ファイル | 説明 |
|----------|------|
| `Credentials.json` | Google Cloud サービスアカウントのJSONキー |
| `.env` | 環境変数（`TALENT_DB_SPREADSHEET_ID`） |

### 3. Google Chrome のインストール

reCAPTCHA回避のため、Google Chrome のインストールが必須です。

### 4. 動作確認

`実行.bat` をダブルクリックして手動実行し、正常に動作することを確認します。

### 5. タスクスケジューラに登録

管理者権限の PowerShell で `タスク登録.ps1` を実行します。

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\タスク登録.ps1
```

## ファイル構成

```
エンゲージ_半自動転記/
├── engage_check_apply.py   # メイン処理
├── SpreadsheetManager.py   # スプレッドシート書き込み
├── constants.py            # 定数（エリア・職種マッピング）
├── Notification/           # 通知関連
│   ├── NotificationManagerClass.py
│   └── config.py
├── Credentials.json        # Google認証（Git管理外）
├── .env                    # 環境変数（Git管理外）
├── 必要パッケージ.txt       # 依存パッケージ
├── 初期セットアップ.bat     # 初期セットアップ用
├── 実行.bat                # タスクスケジューラから実行するバッチ
├── タスク登録.ps1           # タスクスケジューラ登録
└── 使い方.md               # このファイル
```

## 注意事項

- VPS（データセンターIP）では reCAPTCHA が高確率で発生します。2Captcha API キーが必要です。
- Google Chrome は常に最新版を維持してください。
- 実行時間制限は55分に設定されています（1時間間隔のため）。
